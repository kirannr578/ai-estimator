## How to prioritize if Q&A space is limited

Some TAMU System CSPs limit each proposer to a small number of questions (e.g. 5 or 10) to keep the addenda manageable. If a limit applies, prioritize:

1. **Q1** (eligibility) — only if not already confirmed verbally / by email with Joelle
2. **Q2** + **Q3** (schedule / academic window)
3. **Q6** + **Q7** (lab utilities — fume hood, DI water, lab gas, acid waste) — these are the single largest cost-variance drivers
4. **Q11** (hazmat survey availability)
5. **Q18** (insurance limits)

Q22 (page-count limit) and Q20 (evaluation weights) are answered by the CSP package itself; only ask if not stated in writing.
