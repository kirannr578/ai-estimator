- [ ] **Active SAM.gov registration** — not strictly required for TAMU System state work, but keeps the firm visible on federal and partially-funded jobs. `[USER TO FILL: confirm CAGE + expiration]`

- [ ] **Texas Sales & Use Tax Exemption awareness** — TAMU is sales-tax-exempt on materials incorporated into the work; firm needs to be set up to invoice tax-exempt and accept the exemption certificate. (See ASU Attachment E pattern.)
- [ ] **TX HUB status** — firm's own HUB certification if applicable. If non-HUB, the HSP becomes the only HUB-participation path — see `05-hsp-plan.md`. `[USER TO FILL]`
- [ ] **Bonding capacity** — single-project capacity ≥ ~$300K (above the inferred bid envelope) AND aggregate capacity headroom of at least 1× the bid amount above currently-bonded backlog. `[USER TO FILL: confirm with bonding agent — recommend reserving against a $250K envelope]`
- [ ] **Insurance limits per TAMU SGC Article 5** (confirm when CSP arrives; expected from sister TTU System SGC):

  - Builder's Risk — typically owner-furnished on TAMU work; confirm
- [ ] **Texas Contractors Compliance Program / DBE awareness** — for federal-funded portions only (none indicated)
- [ ] **No statewide TX GC license required** (Texas does not license general contractors at the state level); confirm no municipal / county license is required for College Station / Brazos County work

## C. Trade / subcontractor coverage (Notice-scope-driven)

Each row: identify a candidate sub (or self-perform), confirm bonding/insurance acceptable to TAMU, lock pricing.

- [ ] **Demolition (selective interior)** — needs experience with occupied educational buildings, dust containment, after-hours scheduling
- [ ] **Framing / drywall** — light-gauge metal stud + GWB typical
- [ ] **Doors, frames, hardware** — confirm whether existing door / frame stay or replace
- [ ] **Flooring** — VCT / LVT / rubber base typical for educational lab. Chemical-resistant flooring (epoxy / sheet vinyl with heat-welded seams) likely required for lab portion — confirm against drawings
- [ ] **Painting** — institutional-grade interior paint; low-VOC for occupied buildings
- [ ] **Electrical (lab-experienced)** — emergency power, dedicated GFCI circuits for lab benches, possible UPS for tech equipment, lighting (LED retrofit + controls)
- [ ] **HVAC (lab-experienced)** — variable-volume / lab-airflow awareness, fume hood interlock if present, exhaust-stack make-up air balance
- [ ] **Plumbing (lab-experienced)** — eyewash + safety shower, DI water tap, lab gas piping if present, acid-waste piping if present (often duriron or polypro)
- [ ] **Casework / lab-grade millwork** — phenolic resin tops, chemical-resistant epoxy tops, plastic-laminate base cabs are typical
- [ ] **Low-voltage / data / AV** — coordinate with **TAMU Technology Services** (they often self-perform low-voltage in-house and require the GC to provide pathway / rough-in only)
- [ ] **Acoustical ceilings** — typical 2x4 lay-in grid + tile
- [ ] **Fire alarm / sprinkler modifications** — NICET-certified preferred; coordinate with TAMU EHS for shutdown windows. Existing-system tie-in only (single room is unlikely to trigger a whole-floor system upgrade, but confirm).
- [ ] **Lab utility specialty** — DI water generation/distribution, vacuum, lab gases, compressed air. Likely a specialty sub; confirm scope against drawings
- [ ] **Hazmat / lead / asbestos abatement** — older campus buildings may need abatement before demo; TAMU EHS provides existing hazardous-materials survey; confirm need
- [ ] **General requirements / trash / dumpsters / temp services** — TAMU often restricts dumpster location and dictates the loading-dock route; coordinate with Joelle on logistics
