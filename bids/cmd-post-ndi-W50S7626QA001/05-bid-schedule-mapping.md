# Bid schedule / CLIN-to-cost mapping

The solicitation does **not** include a separately attached "bid schedule" PDF (unlike, e.g., the USFWS sibling bid which had its own Bid Schedule attachment). Section B of the SF 1442 defines the formal CLIN structure, and Section L paragraph (a)(1) specifies that the Offeror submits **a firm-fixed-price proposal, clearly separated into the following Contract Line-Item Numbers (CLINs):**

> -CLIN 0001: Building 1672 Command Post Relocation
> -CLIN 0002: Building 1675 Nondestructive Inspection (NDI) Room

Each CLIN is **Quantity 1 Job, Firm Fixed Price**, priced **independently** (so the Government can award one CLIN without the other).

This document maps the CLIN structure to the trade-by-trade scope from `04-scope-of-work.md` and identifies the cost-DB candidates and sub-quote gaps for each.

Document this as a contingency line in the offer's internal-cost tab; do **not** disclose to the Government separately (it's part of the FFP).

## C. Pricing-rollup structure (internal — not delivered to Government separately)

```
Direct cost (labor + materials + sub) ............. base
+ Field overhead (supervision, layout, crane, FOD, daily cleanup, fence) ..... ~7-10%
+ General conditions (PM, safety, QC, submittals, badging coordination) ...... ~6-9%
+ Insurance (GL + Auto + WC) — small project loading .......................... ~2-3%
+ Bid bond + P&P bonds ........................................................ ~1.5-2.5%
+ Contingency (CLIN 0001: secure-area scope unknowns) ......................... ~3-5%
+ Contingency (CLIN 0002: NDI Sheet 2 + AFFF residue unknowns) ................ ~8-12% (CLIN 0002 only)
+ Home-office overhead ........................................................ ~4-6%
+ Profit ...................................................................... ~6-9%
= Bid price (per CLIN, FFP)
```

For **comparative-evaluation** procurements (NOT LPTA), profit can sit at the upper end of the range as long as past-performance + technical strength carry the comparative ranking. See `06-evaluation-strategy.md`.

## D. Bid-envelope estimate (top-down sanity check)

The user's original triage assumed "$50–100K LPTA" — that was a guess. With drawings + SOW + photos in hand, a defensible **floor-to-ceiling envelope** for a 100% small business set-aside ANG interior renovation in Tarrant County:

| Building | Approx renovation area (SF) — **VERIFY at site visit** | Federal interior reno benchmark | Range estimate |
|---|---|---|---|
| B1672 Command Post (incl. mantrap, secure doors, comm rough-in, full HVAC + electrical mod) | ~1500–2500 SF | $180–$280/SF for **secure-area** federal interior renovation (10–25% premium over standard reno) | **$270K – $700K** |
| B1675 NDI Room (AFFF conversion, standard interior reno + light electrical/mech mod, possible PFAS-residue handling) | ~400–800 SF | $130–$220/SF for standard federal interior renovation + AFFF environmental contingency | **$50K – $175K** |
| **Combined floor / mid / ceiling** | | | **~$320K (floor) / $475K (mid) / $875K (ceiling)** |

> ⚠️ **Wide range** because we don't have field-verified SF, the existing fire/HVAC system condition, or the actual Sheet 2 NDI scope. **The user's original $50–100K LPTA framing was almost certainly low by 4–10x.** The bond-broker should be told a $500K reservation envelope (mid + 30% headroom).
