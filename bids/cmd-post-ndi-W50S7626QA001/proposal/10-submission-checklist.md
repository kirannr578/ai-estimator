## E. Attachment size

| Item | Approx PDF size |
|---|---|
| SF 1442 (signed scan, 2 pages) | 2–4 MB |
| Schedule of Prices | < 1 MB |
| Subfactor 1 list | 1–2 MB |
| Subfactor 2 schedule (Gantt) | 1–2 MB |
| Bid Bond SF 24 (scan) | 1–2 MB |
| Past Performance | 2–4 MB |
| Bondability letter | < 1 MB |
| Key personnel | 1–2 MB |
| **Total** | **~8–18 MB** |

> Most enterprise email systems (especially `.mil` accounts) accept attachments up to 25 MB. If total > 20 MB, use a single ZIP file or merged PDF to consolidate.
