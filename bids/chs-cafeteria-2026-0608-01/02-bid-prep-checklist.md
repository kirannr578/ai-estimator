# CHS Cafeteria Serving Line Renovation — Bid-Prep Checklist (TX State CSP / RFCSP w/ HSP)

> Pulled from common denominator of [`bids/tamu-harrington-2025-06813/02-bid-prep-checklist.md`](../../tamu-harrington-2025-06813/02-bid-prep-checklist.md) and [`bids/angelo-state-carr-efa-26-007/`](../../angelo-state-carr-efa-26-007/).

## Pre-pricing (Day 1–5 after NOP capture)

- [ ] CSP package downloaded from e-Builder / SharePoint per NOP — drawings, project manual, sample CSA, HSP form, pricing form, bid-bond form, addenda
- [ ] Read NOP + Project Manual + all addenda end-to-end
- [ ] TX HUB cert verified current (or recertify — see [`firm/compliance/README.md`](../../../firm/compliance/README.md))
- [ ] TX franchise-tax good standing verified via TX Taxable Entity Search
- [ ] TX CMBL enrollment verified
- [ ] Confirm pre-response meeting attendance — mandatory or recommended? RSVP submitted
- [ ] Confirm `2026-06-03 10:00 CT` and start RFI drafting
- [ ] **Confirm separate HSP deadline** if applicable (TTUS bids often have HSP due 1–3 days after proposal)
- [ ] Pull `Johnson` County prevailing wage `Davis-Bacon Prevailing Wage Rates for Johnson County (SAM.gov; attached to RFCSP)`; transcribe applicable trade rates
- [ ] Bondability commitment letter ordered from surety
- [ ] Bid bond (5% of proposal) ordered
- [ ] P&P bond commitment letter ordered for award
- [ ] Current GL COI pulled — verify Umbrella ≥ $5M for institutional SGCs
- [ ] Current WC + Auto COIs pulled
- [ ] Past-performance picks selected per [`firm/firm-profile.json → past_project_selection_rules`](../../../firm/firm-profile.json) for this bid type
- [ ] Past-perf owner-side reference contacts confirmed

## HSP outreach (Day 1 onward — runs in parallel)

- [ ] HUB-certified sub list pulled from TX CMBL HUB Search for each trade (≥ 5 per trade)
- [ ] Standardized sub solicitation packet sent to ≥ 5 HUBs per trade
- [ ] Outreach log started (`local/hub-outreach-log.csv`) — date, trade, sub, method, response
- [ ] ≥ 2 HUB-publication advertisements placed
- [ ] HUB Discretionary Contracting Forum / `Cleburne ISD` HUB Operations contacted if applicable
- [ ] Daily outreach status reviewed by PM

## Pre-response meeting + site visit (Day 5–10)

- [ ] Attend pre-response meeting; take notes; obtain published Q&A
- [ ] Attend site walk; capture site-visit shopping list per scope-template
- [ ] Photo log filed; measurements verified against drawings
- [ ] Sign-in sheet copy obtained

## Sub procurement (Day 5–22)

- [ ] Sub solicitations issued (≥ 3 quotes per major trade)
- [ ] Sub vetting checklist (per [`firm/proposal-library/boilerplate/subcontractor-management.md`](../../../firm/proposal-library/boilerplate/subcontractor-management.md)) underway per sub
- [ ] HUB-status (cert # + expiration) verified for each HUB sub
- [ ] OSHA training cards (10 + 30) collected per sub foreman

## Pricing assembly (Day 15–25)

- [ ] Direct cost rolled up (subs + materials + self-perform labor + materials)
- [ ] General conditions priced (`49 (NTP 2026-06-16 → 2026-08-03 final completion)` cal day footprint)
- [ ] Bond + insurance loaded as percentage
- [ ] TX prevailing-wage uplift applied if firm labor < prevailing
- [ ] Contingency applied (5–7% institutional; 10% if drawings thin)
- [ ] Overhead applied (8–12%)
- [ ] Profit applied (5–7%)
- [ ] Lump-sum cross-checked against `265,000`–`325,000` magnitude band
- [ ] Unit prices set at full retail (change-order ceiling)
- [ ] Alternates priced per CSP request (do NOT add unsolicited)
- [ ] Allowances carried separately per CSP instructions
- [ ] HSP commitment rolled up — verify ≥ statewide goal or project goal

## Proposal assembly (Day 20–28)

### Front matter

- [ ] Cover letter / executive summary per [`firm/proposal-library/exec-summary-archetypes/texas-state-csp.md`](../../../firm/proposal-library/exec-summary-archetypes/texas-state-csp.md)
- [ ] Proposal Form / Execution of Offer signed (per CSP §00 42 13 typ)
- [ ] CIQ (Tex. Local Gov't Code Ch. 176) signed
- [ ] HB 1295 Certificate of Interested Parties filed with TEC (if applicable)
- [ ] Non-collusion affidavit signed (if required)
- [ ] Resident-bidder attestation signed (if required)
- [ ] Addenda acknowledgments listed (`{{ADDENDA_LIST}}`)
- [ ] W-9 current calendar year, signed

### Pricing

- [ ] Pricing form complete — lump-sum + unit prices + alternates
- [ ] Bid bond (5%) attached, original
- [ ] Surety commitment letter for P&P bonds attached
- [ ] Insurance certificates attached, with `Cleburne Independent School District` named as Additional Insured + waiver + primary/non-contributory

### Technical / Qualifications / Team

- [ ] Executive summary (1 page, paste from archetype)
- [ ] Technical approach (multi-page; pull from [`bids/tamu-harrington-2025-06813/proposal/02-technical-approach.md`](../../tamu-harrington-2025-06813/proposal/02-technical-approach.md) pattern)
- [ ] Project team org chart + key-personnel résumés from [`firm/proposal-library/key-personnel/`](../../../firm/proposal-library/key-personnel/)
- [ ] Past performance (3 picks per `past_project_selection_rules`)
- [ ] Schedule narrative + CPM Gantt per [`firm/proposal-library/boilerplate/schedule-narrative-skeleton.md`](../../../firm/proposal-library/boilerplate/schedule-narrative-skeleton.md)
- [ ] QC plan per [`firm/proposal-library/boilerplate/qa-qc-plan-one-pager.md`](../../../firm/proposal-library/boilerplate/qa-qc-plan-one-pager.md)
- [ ] Safety plan per [`firm/proposal-library/boilerplate/safety-plan-one-pager.md`](../../../firm/proposal-library/boilerplate/safety-plan-one-pager.md)
- [ ] Sub management + HSP narrative per [`firm/proposal-library/boilerplate/subcontractor-management.md`](../../../firm/proposal-library/boilerplate/subcontractor-management.md)
- [ ] Closeout plan per [`firm/proposal-library/boilerplate/closeout-plan.md`](../../../firm/proposal-library/boilerplate/closeout-plan.md) (if requested)
- [ ] Communication plan per [`firm/proposal-library/boilerplate/communication-plan.md`](../../../firm/proposal-library/boilerplate/communication-plan.md) (if requested)

### HSP

- [ ] HSP completed on agency-specific form (NOT a prior bid's form)
- [ ] Per-trade HSP allocation table (see `05-hsp-plan.md`)
- [ ] GFE binder compiled — outreach log + ad evidence + Q&A responses + non-response documentation
- [ ] Self-performing justification written
- [ ] HSP signed by signer + dated
- [ ] If HSP has separate deadline → calendar reminder set

### Submission package

- [ ] Sealed-envelope exterior labeling exactly per CSP instructions
- [ ] Number of copies per CSP
- [ ] Final QC review by second BPC team member
- [ ] All `[USER TO FILL]` markers closed via [`firm/_scripts/scan_placeholders.py`](../../../firm/_scripts/scan_placeholders.py)

## Submission (Day 28–30)

- [ ] Hand-deliver or courier to `Cleburne ISD, Attn: Stefanie McBroom, 505 N. Ridgeway Dr. Ste. 100, Cleburne, TX 76033` before `2026-06-08` @ `14:00`
- [ ] Receipt signed at delivery
- [ ] Receipt timestamp filed
- [ ] **HSP submitted on its separate deadline** if applicable
- [ ] Public opening attended (in person or via Teams livestream if `Cleburne ISD` streams)

## Post-submission

- [ ] BAFO / clarifications response if requested by `Cleburne ISD`
- [ ] Reference verification answered (if `Cleburne ISD` calls references)
- [ ] Monitor ESBD for award notice
- [ ] If awarded → trigger post-award checklist
- [ ] If lost → request scoring detail; document lessons learned
