## Notes for the user before sending

- **Substitute the bracketed fields** with the firm-specific values.
- **Confirm the bonding agent's working capacity** before sending the request — if the firm is bonded by `[USER TO FILL]` already, ensure the agent has been pre-warned about the $250K-ish reserve.
- **If multiple bonding agents are in rotation,** send to the one with the strongest {{AGENCY_SHORT}} System work history ({{AGENCY_SHORT}} has informal preferences for sureties with TX higher-ed experience).
- **The exact bond-form requirement** is part of the CSP package. Until G2 closes, the bonding agent is working off a placeholder. Re-confirm the bond form on G2 close and forward to the agent same-day.
- **Bid-bond cost** is typically a flat $100–$300 issuing fee for a sub-$500K bond. P&P-bond cost is typically 0.6–1.2% of contract value, collected from the contractor at award (not at bid).
- **If the agent cannot accommodate** the timeline or capacity, escalate immediately — bonding-agent issues are a hard go/no-go gate.
