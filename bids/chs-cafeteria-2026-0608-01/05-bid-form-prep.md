## 6. HB 1295 — Certificate of Interested Parties (Tex. Gov't Code §2252.908)

Required for any contract ≥ $1M OR any contract that requires legislative approval. File **online with the Texas Ethics Commission** at <https://www.ethics.state.tx.us/Forms/Form_1295.htm>. Process:

1. BPC files online — captures contract identifying number + counterparty (agency)
2. TEC issues a unique 1295 Certification Number
3. Print the certified PDF
4. Sign + notarize the signature page
5. Submit the certified PDF + notarized signature to the agency at award

Pre-bid: file a placeholder (with the procurement number) so BPC can quickly submit at award if won.
