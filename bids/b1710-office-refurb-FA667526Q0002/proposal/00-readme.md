Per `04-evaluation-strategy.md`, the CO is most likely running **comparative evaluation per FAR 12.203(c)(2)**. The attachments map to evaluation dimensions:

- **Responsiveness / responsibility** — Attachments 1, 5, 6, 7 (SF 1449 signed, SAM active, payment-bond capacity, COI)
- **Price** — Attachment 2
- **Technical acceptability** — Attachment 3 (equipment / labor / schedule)
- **Past performance** — Attachment 4 (3-pack)
- **Compliance** — Attachment 8 (Buy American), implicit in 1 (Davis-Bacon flow-down)
