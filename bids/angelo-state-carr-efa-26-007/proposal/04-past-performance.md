# Past performance — Carr EFA Dressing Room Renovation

> Target length: 1–2 pages. Three project references are the floor; five is typical for a TTUS RFCSP. Each project gets a table header for at-a-glance review plus a 1-paragraph narrative covering relevance to Carr EFA.

> **Project-selection strategy (per `../06-evaluation-strategy.md` § B):** prioritize, in order:
> 1. Prior TTUS member-institution work (TTU main, ASU, ASU-San Angelo, ASU-El Paso, AS-Health Sciences Center). Owner familiarity with the firm is the single strongest scoring signal.
> 2. Prior TX higher-ed renovation under $1M (UT System, TAMU System, other TX state schools).
> 3. Performing-arts or theater-support renovation (any owner, any state).
> 4. Active-building / occupied-space institutional renovation (any owner, any sector — hospital, courthouse, school).
> 5. Any sub-$1M institutional renovation with strong MEP-replacement scope.
>
> The firm needs three solid references; if the firm has all five categories, surface them in this order. If the firm has only one or two categories, fill the remaining slots with the strongest available institutional renovation references.

## Reference contact instructions

- Verify each owner contact's **current** phone and email **before** submission. Stale references with disconnected numbers are interpreted as inattention by TTUS evaluators.
- Notify each reference contact ≥48 hours in advance that an ASU evaluator may call. A reference that is "caught cold" rates worse than a reference that has been primed to expect the call.
- If the firm has no prior performing-arts renovation, lead the narrative on each of References #1–#3 with the closest analogous experience: occupied-building MEP replacement, ADA-alterations compliance, after-hours work in active institutional buildings, owner-furnished coordination on access control / IT systems / FF&E.

## Cross-reference to evaluation strategy

Per `../06-evaluation-strategy.md` § A, past performance / qualifications typically carries **15–25% of the total evaluation score**. This is the second-largest scoring factor after price and is the easiest to differentiate on. The narrative for each project should directly answer the question: **"Has this firm done a project that looks like Carr EFA, and did it go well?"**
