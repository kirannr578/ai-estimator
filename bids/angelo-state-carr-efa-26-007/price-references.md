# Price references — Carr EFA Dressing Room Renovation

> Benchmarks + comparable past-project ranges for sizing the Carr EFA bid. These are national / regional averages adjusted for context — they are NOT a substitute for actual sub-quotes against actual drawings.

---

## A. Headline $/SF range we landed on

| Scenario | $/SF range | Total at 2,500 SF assumed footprint |
|---|---|---|
| **Light dressing-room reno** — finishes + paint + light electrical + minor plumbing modifications, no full MEP system replacement | $90 – $140 / SF | $225K – $350K |
| **Mid-scope reno** — finishes + paint + new lighting + selective MEP (HVAC retrim, plumbing fixture replacement) | $130 – $200 / SF | $325K – $500K |
| **Full reno with complete MEP system replacement** ← matches the Carr EFA RFCSP narrative | **$165 – $260 / SF** | **$413K – $650K** |
| **Heavy reno with showers / wet zones + theater-grade millwork** (if dressing rooms include showers and full makeup-vanity millwork) | $200 – $320 / SF | $500K – $800K |

**Working envelope for this bid: $425K – $560K (midpoint ~$485K), against the $500K published budget.**

The high end of the range pushes above the published $500K budget — the $500K cap should focus the bid on the low-to-mid end of the "Full reno" scenario rather than the "Heavy reno" scenario, OR Drive a value-engineering conversation if the heavy-reno scope is what's actually drawn.

## B. Why those numbers

The Carr EFA RFCSP narrative explicitly calls for "complete demolition and replacement of all interior finishes, mechanical systems, plumbing systems, electrical systems, along with all new IT infrastructure, and ADA code elements." That language matches the "Full reno with complete MEP system replacement" row above.

Cost-stack expectations (rough percentages of subtotal for a full-MEP-replacement reno, before contingency/OH/profit):

| Division | % of subtotal | $/SF at midpoint $200/SF |
|---|---|---|
| Div 01 General Requirements | 12–18% | $24–$36 |
| Div 02 Demo | 6–10% | $12–$20 |
| Div 04 Masonry (brick restoration) | 1–3% | $2–$6 |
| Div 06 Carpentry + millwork | 8–14% | $16–$28 |
| Div 08 Doors + Sargent locks + CBORD prep | 4–7% | $8–$14 |
| Div 09 Finishes (drywall, ACT, flooring, base, paint) | 14–22% | $28–$44 |
| Div 10 Specialties | 1–3% | $2–$6 |
| Div 22 Plumbing | 8–12% | $16–$24 |
| Div 23 HVAC (complete replacement) | 14–22% | $28–$44 |
| Div 26 Electrical (complete replacement + lighting) | 14–22% | $28–$44 |
| Div 27 LV pathway | 1–2% | $2–$4 |
| Div 28 Fire alarm relocation | 1–2% | $2–$4 |
| **Sum** | **~100%** | **~$200/SF (target)** |

Add contingency 8%, OH 10%, profit 6% → **~$200 × 1.08 × 1.10 × 1.06 ≈ $252/SF all-in.** Round to $250/SF midpoint for the headline range.

## D. Owner-furnished items that pull cost OUT of our pricing

| Owner-furnished item | Typical $/SF if GC carried | Net deduct from our scope vs full-scope baseline |
|---|---|---|
| Fire suppression (sprinkler relocation + branch mods on a full reno) | $5–$12/SF | ~$15K–$30K saved on our number |
| AV / data cabling + jacks + termination | $4–$8/SF | ~$10K–$20K saved |
| FF&E lockers / costume racks / makeup chairs | $8–$25/SF | ~$20K–$60K saved |
| Equipment install (post-construction equipment install) | $2–$6/SF | ~$5K–$15K saved |
| CBORD readers + controllers + electric strikes (devices only) | $1–$3/SF | ~$2K–$7K saved |
| **Sum of owner-furnished deducts vs full-scope baseline** | **~$20–$54/SF** | **~$50K–$130K** |

Net: our scope sits at roughly **70–80% of the all-in renovation cost** with the rest being owner-furnished. This is a meaningful tailwind on the bid — we can be confident the published $500K budget includes owner-furnished items separately AND/OR represents only the GC-scope portion of a larger total project investment.

## E. Three pricing scenarios to test against drawings

When drawings arrive, run the takeoff against each scenario and pick the one that best matches the drawing scope:

### Scenario A — Light reno (lower bound)
- Assumes: 2,000 SF, finishes + light MEP only, no showers, owner-furnished list as described
- Target subtotal: ~$220K × 1.08 × 1.10 × 1.06 = **~$275K all-in**
- This is below the $500K budget — leaves significant owner discretion for additional scope or contingency

### Scenario B — Mid reno (most likely)
- Assumes: 2,500 SF, full MEP replacement, no showers, owner-furnished list as described
- Target subtotal: ~$385K × 1.08 × 1.10 × 1.06 = **~$485K all-in**
- This matches the $500K budget closely; sits at the midpoint of our headline range

### Scenario C — Heavy reno (upper bound)
- Assumes: 3,000 SF, full MEP replacement + showers + theater-grade millwork, owner-furnished list as described
- Target subtotal: ~$540K × 1.08 × 1.10 × 1.06 = **~$680K all-in**
- This exceeds the $500K budget — would require either a budget revision, scope reduction, or aggressive value engineering. Surface to user + Samuel immediately if drawings indicate Scenario C scope at $500K budget.
