## What if the bonding agent can't issue both Performance + Payment bonds?

**Alternative #1 — Irrevocable Letter of Credit (ILC):**
- ILC from BPC's banking relationship for 100% of contract price
- Replaces Performance + Payment bond in some TAMU System contexts
- ILC must be from a bank acceptable to TAMU System (typically TAMU's banking relationships)

**Alternative #2 — Subdivide bonds:**
- Performance Bond from Surety A
- Payment Bond from Surety B
- Both Treasury Circular 570 listed

## Bond capacity verification needed

Per `firm/firm-profile.json`, BPC has demonstrated $1,000,000 single-project capacity (Lavon RV Park). For Wehner (estimated $145K-$175K), this is well within capacity. Bonding agent should confirm:

- [ ] Single-project bond capacity ≥ $200,000
- [ ] Aggregate bond capacity ≥ $1,500,000 (sufficient for ongoing portfolio + Wehner)
- [ ] Bonded project history: 0 defaults / claims
- [ ] Surety financial rating: A.M. Best B+ or better

## Common bonding-letter errors

| Error | Avoidance |
|---|---|
| Letter older than 30 days at bid submission | Request letter dated within 7-14 days of submission |
| Surety not Treasury Circular 570 listed | TAMU System may not accept; verify before requesting |
| Letter doesn't mention specific Sol # | Include "Sol SSC-2025-06871" in the letter explicitly |
| Letter doesn't state bond %/scope | Letter must specify 100% Performance + 100% Payment |
| Letter doesn't address PoP + 1 year | Standard but verify |

## Submission

The bonding commitment letter is attached to the proposal packet as a PDF. Multiple letters (one each for Performance + Payment) are acceptable; a single letter covering both is preferred for clarity.
