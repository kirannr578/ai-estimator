## 6. Procurement vehicle — CSP under TAMU System

This is a **Competitive Sealed Proposal** (CSP) per Texas Education Code Ch. 51 and Government Code Ch. 2269. CSP is a "best value" procurement (not lowest price); the evaluation criteria typically include:

- Price (typically 40-60% of total score)
- Past performance (typically 15-25%)
- Schedule / construction approach (typically 10-15%)
- Project team / experience (typically 5-10%)
- HUB participation / HSP (typically 5-10%)
- Safety record / EMR (typically 5%)

Actual weights vary per TAMU System project; will be in the CSP package. See `06-evaluation-strategy.md` for the assumed scoring model and BPC's positioning.
