## 1. HSP framework (TAMU System pattern)

TAMU System CSPs typically use one of two HSP frameworks:

| Framework | Form | Scoring |
|---|---|---|
| **TX State Subcontracting Plan (Rev. 5/26)** — current standard | CSP Section 00 45 17.1 | Used by TAMU Harrington (sister sol); typically integrated into VetHUB framework |
| TAMU-specific HSP overlay | Sometimes layered on top | If applicable, will be in the CSP package |

**Both** frameworks score:
- Total HUB participation as % of contract value
- Number of HUB subs solicited
- Documentation of outreach (dates, methods, responses)
- "Good Faith Effort" (GFE) — required even if HUB goal isn't met

## 6. HSP form-fill placeholder mapping

When the Section 00 45 17.1 form (TX State Subcontracting Plan Rev. 5/26 OR VetHUB framework) is in hand, the fields below will need to be populated:

- **Section 1**: Identify whether good-faith effort was achieved OR HUB goal was achieved
- **Section 2**: List of HUB subs solicited (3+ per trade typical)
- **Section 3**: List of HUB subs that responded with quotes
- **Section 4**: List of HUB subs that received subcontract awards (or commitment for award upon CSP award)
- **Section 5**: Total HUB participation % of contract value
- **Section 6**: Narrative justification for good-faith effort if goal not met
- **Section 7**: Monthly HUB Progress Assessment Report (PAR) commitment for post-award

The full form-fill guide is in `proposal/09-hsp-form-fill-guide.md`.

## 7. Documentation requirements

- Trade scope solicited

- Quote amount (if quoted)
- Subcontract commitment (yes/no)

All documentation maintained for 3+ years post-award per TAMU SGC.
