# Exec Summary Archetype — Texas State CSP / RFCSP with HSP

> **Source bids mined:** [`bids/tamu-harrington-2025-06813/proposal/01-executive-summary.md`](../../../bids/tamu-harrington-2025-06813/proposal/01-executive-summary.md), [`bids/angelo-state-carr-efa-26-007/proposal/01-executive-summary.md`](../../../bids/angelo-state-carr-efa-26-007/proposal/01-executive-summary.md)
> **Use:** Cover letter / Executive Summary at the front of the bound proposal. **One page.** State CSPs **do score qualitative factors** (unlike LPTA), so the executive summary is the moment to frame project understanding, team, and HSP commitment in a way that primes the evaluator.

## How to trim per RFP

- If the CSP package limits the cover letter to a single page, **drop the BONDS + INSURANCE block** (lives in Section 04, Qualifications) and tighten the QUALIFICATIONS paragraph.
- If the CSP scores executive summary content (rare), keep all four substantive blocks (PROJECT UNDERSTANDING, QUALIFICATIONS, HSP COMMITMENT, TEAM + SCHEDULE) and tighten the cover-letter formalities.
- If the agency does **not** require HSP, drop the HSP COMMITMENT block.
- If the agency uses a different SGC than the TX UGSC (e.g. a city or ISD with its own conditions), update the ACKNOWLEDGMENTS block to reference the right document.

## What to KEEP OUT

- Detailed schedule (Gantt-style breakdown belongs in Section 05, Schedule)
- Pricing breakdown (lump-sum only here; CLIN-style breakdown in the Proposal Form)
- Per-trade subcontractor names (sub list, if required, is its own section)
- Marketing language ("award-winning", "industry-leading", "best-in-class") — TX evaluators distrust it
- References to subcontractor work as if BPC self-performed
- Anything that takes the proposal past one page (state evaluators read 5–15 of these per CSP; respect their time)
