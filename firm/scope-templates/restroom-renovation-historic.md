# Scope Template — Restroom Renovation in Historic Structures

> **Source context:** SAAN San Juan watchlist entry ([`bids/_WATCHLIST/saan-san-juan-restrooms-140P1226Q0025.md`](../../bids/_WATCHLIST/saan-san-juan-restrooms-140P1226Q0025.md)) + general NPS / NRHP-listed building precedents. NPS Preservation Briefs 4 (Roofs), 24 (HVAC), 35 (Tax-Act standards), and the Secretary of the Interior's Standards for Rehabilitation are the controlling references.
>
> **Hard differentiator from a standard restroom renovation:** every demo decision, every fixture replacement, every penetration through historic fabric requires SHPO (State Historic Preservation Office) — or in NPS units, the regional Cultural Resources Manager — review. Schedule is dominated by review cycles, not by trade duration.
>
> **Typical CSI divisions touched:** 01, 02 (very careful), 06, 08, 09 (historic tile + plaster + paint), 10 (toilet accessories + partitions), 22 (plumbing), 23 (ventilation), 26 (electrical).

## 1. Scope outline (standard sequencing)

| # | Phase | Trade(s) | Typical duration (2-stall single-gender restroom in historic building) |
|---|---|---|---|
| 1 | Pre-construction historic-conditions assessment + photo-documentation | Preservation consultant + GC | 1–2 weeks |
| 2 | SHPO / Section 106 pre-construction review submittal + approval | Preservation consultant + agency | 4–8 weeks (slow — usually critical path) |
| 3 | Mobilization + dust/vibration protection of adjacent historic fabric (custom envelope, plastic + plywood crating around adjacent millwork, vibration monitor) | GC + preservation contractor | 3–5 days |
| 4 | Selective demolition — hand-tools only near historic fabric; document every removed element + retain for re-use or museum collection | Preservation demo sub | 1–2 weeks |
| 5 | MEP rough-in (challenge: routing supply + DWV + venting through historic envelope without exposed runs) | Plumber + electrician with retrofit experience | 1–2 weeks |
| 6 | Fixtures + finishes install (period-appropriate per SHPO approval) | Plumber + tile setter | 1–2 weeks |
| 7 | Toilet accessories + partitions (ADA-compliant; period-appropriate finish) | Carpenter | 3–5 days |
| 8 | Punch + cleaning (no harsh chemicals on historic plaster / wood / tile) | GC | 3–5 days |
| 9 | SHPO post-construction sign-off + as-built submittal | Preservation consultant | 1–2 weeks |
| **Total** | | | **10–18 weeks (review cycles dominate)** |

## 2. Standard takeoff-item list

### 2.1 Demolition (extra-careful in historic buildings)

```
DEMO (NPS / SHPO Section 106 requires photo-documentation BEFORE every removal)
  □ Existing toilet fixtures (each) — verify whether period-appropriate originals retain salvage value
  □ Existing lavatory fixtures (each) — same
  □ Existing toilet partitions (each + LF)
  □ Existing wall tile (SF) — IF historic (subway, hex floor, 4x4 wall), bag + retain samples; identify replacement source
  □ Existing floor tile (SF) — same; original hex tile commonly retained as museum collection
  □ Existing plumbing fixtures + valves + trim (each)
  □ Existing supply + DWV + vent piping (LF) — CAREFUL routing through historic envelope
  □ Existing toilet accessories (each)
  □ Existing finish-side ceiling (SF) — IF dropped acoustic, demo; IF historic plaster ceiling, scope ONLY for SHPO-approved openings
  □ NO removal of historic millwork, trim, doors, or fixtures without explicit SHPO approval
  □ Vibration monitoring during demo (typ Larcor or similar, daily logs)
  □ Hand-tools only within 6' of historic adjacent fabric
```

### 2.2 Period-appropriate fixtures (SHPO-approved)

```
FIXTURES (verify EACH selection against SHPO submittal)
  □ Toilet — wall-hung, period-appropriate (typ Kohler Memoirs, American Standard Boulevard, or restoration-line equivalent); WaterSense 1.28 gpf
  □ Lavatory — period-appropriate (pedestal OR wall-hung console; typ Kohler Memoirs, restoration-supply houses like Vintage Plumbing); 0.5 gpm aerator
  □ Faucet trim — chrome OR polished brass OR satin nickel (period-matched); cross-handle or lever (verify ADA)
  □ Toilet partition — solid plastic OR powder-coated steel; period-appropriate color (typ off-white, sage, cream)
  □ Grab bars — stainless OR chrome (NOT brushed nickel — too modern for most historic settings); 1.5" dia per ADA
  □ Mirror — beveled-edge OR plain-edge, framed OR unframed per SHPO; verify ADA mounting
```

### 2.3 ADA upgrades within historic constraint

```
ADA (critical — ADA + Section 106 must BOTH be satisfied; sometimes alternative-compliance path required)
  □ 60" turn radius at minimum 1 fixture (typically requires removing 1 stall to gain space)
  □ Transfer space at WC (5' min on each side OR end-stall transfer config)
  □ Grab bars rear + side per ADA 604.5
  □ Lav clearance + knee space per ADA 606
  □ Hardware reach + force per ADA 309
  □ Doorway 32" clear minimum (historic doors typ 30" — alternative-compliance may apply if dimension cannot be widened without damage to historic envelope)
  □ Signage (Braille + tactile) on latch side at 60" AFF
```

### 2.4 Finishes (period-appropriate)

```
FLOORING
  □ Hex floor tile (1" or 2" hex) — typ Daltile, American Olean, restoration houses
  □ OR subway floor tile (3x6 or 4x4) — alt for early-1900s era
  □ OR salvaged + cleaned original tile re-set with matching grout
  □ Tile sealer (penetrating, not topical)
WALLS
  □ Subway tile (3x6 white, beveled-edge OR flat-edge) typ to 4'-6' AFF
  □ Cap rail at top of wainscot (bullnose OR cove tile)
  □ Above wainscot: plaster patch + lime paint OR period-appropriate latex (matte finish; NOT semi-gloss in historic settings)
CEILINGS
  □ Plaster repair + patch (SF) — match existing thickness + texture
  □ Lime paint OR period-appropriate latex (matte)
```

## 4. Typical subcontractors

| Trade | Type | Sub source | Notes |
|---|---|---|---|
| Preservation consultant | Section 106 submittals + SHPO liaison + photo-doc + as-builts | Local architectural historian / preservation architect (state HPO maintains directory); for NPS units, the park's Cultural Resources Manager is often the in-house counterpart | **NOT optional on NRHP-listed or NPS-managed properties** |
| Preservation contractor (demo + envelope-adjacent) | Hand-demo + protection of adjacent fabric | Specialty preservation GC; smaller pool than commercial market; verify NPS or SHPO past performance | Premium 30–60% over commercial demo sub |
| Plumber (retrofit experience) | Routing through historic envelope without exposed runs | Local commercial plumber with documented restoration projects | Standard plumber will damage fabric — verify portfolio |
| Tile setter (period work) | Subway + hex pattern + period grout color | Specialty tile sub; many ceramic-tile installers cannot match historic pattern + period grout | Mock-up panel required by most SHPOs before full install |
| Plaster contractor | Lath + plaster patch + repair | Specialty plaster sub — most drywall subs CANNOT do plaster; small pool | Pre-screen against past NPS / SHPO work |
| Electrician (low-impact routing) | Concealed conduit through historic envelope; period-appropriate fixtures | Commercial electrician with restoration experience | |

## 5. Common alternates / VE opportunities

| Item | Base | Alternate | Trade-off |
|---|---|---|---|
| Salvage existing original tile vs new period-appropriate | Salvage + re-set | New period-appropriate | Salvage saves on tile material but adds labor; SHPO usually prefers salvage |
| Plaster patch vs drywall infill | Plaster patch (period-correct) | Drywall infill flush with plaster | Drywall is 60–70% cheaper but SHPO frequently rejects |
| Pedestal lav vs wall-hung console | Pedestal | Wall-hung console | Wall-hung is easier ADA compliance but pedestal is more period-correct for some eras |
| ADA path: enlarge restroom vs alternative compliance | Enlarge (requires SHPO approval for envelope change) | Alternative compliance path | Alt-compliance saves construction cost but requires longer Section 106 process |
| Toilet partition material | Solid plastic | Powder-coated steel | Steel is more period-correct for pre-WWII buildings; plastic for post-war |
| Fixture finish | Chrome | Polished brass | Brass is more period-correct pre-1940; chrome thereafter; verify SHPO preference |

## 6. Schedule duration ranges

| Project size | Typical duration | Notes |
|---|---|---|
| Single 1-2 stall restroom in NRHP building | 10–14 weeks | SHPO review cycle is the bottleneck, not construction |
| Multiple restrooms in same NPS unit (e.g., visitor-center men's + women's + family) | 14–18 weeks | Can parallel-construct after single review cycle |
| Multi-building NPS project | 18–26 weeks | Multiple SHPO submittals; phasing for visitor impact |
| Same scope WITHOUT historic constraint | 4–8 weeks | Reference point — historic premium is 50–100% on schedule |

**Critical path:** SHPO / Section 106 review. Standard turn-around is 30 days per FAQ; commonly 60+ days in practice. Plan submittals 2-3 review cycles deep (pre-construction approval; mid-construction Modification 1; final acceptance).

## 7. Code references

- **Secretary of the Interior's Standards for Rehabilitation** (36 CFR 67) — the controlling standard
- **NPS Preservation Briefs:**
  - **Brief 18** (Rehabilitating Interiors in Historic Buildings) — directly applicable
  - **Brief 21** (Repairing Historic Flat Plaster) — for ceiling/wall plaster
  - **Brief 23** (Preserving Historic Cast Iron) — for original cast-iron lavs / drain pipes
  - **Brief 32** (Making Historic Properties Accessible) — ADA-within-historic guidance
- **National Historic Preservation Act (NHPA) Section 106** — federal undertaking review
- **ADA Standards for Accessible Design** + ADA Title II / Title III as applicable
- **State Historic Preservation Office (SHPO)** — state-specific submittal forms (TX: THC; varies by state)
- **IBC + IPC + IECC** — modern code overlay (note: historic exemption may apply per Section 3411 IBC)
- **WaterSense** specification (not code-required but increasingly mandated by federal procurement)

## 8. Common pitfalls

1. **Preservation consultant under-scoped.** Standard estimate uses commercial restroom consultant fee ($2K–$5K); historic restroom needs $8K–$25K. This is the #1 estimating miss on historic work.
2. **SHPO review cycle assumed at 30 days.** Plan for 60–90 days per submittal; build float into schedule narrative.
3. **Demo sub not vetted for historic work.** Standard demo subs use vibratory tools that crack adjacent plaster. Pre-qualify against actual NPS / SHPO past performance.
4. **Modern fixture brand spec'd.** Mod-style fixtures (e.g., Toto Soiree, Duravit Vero) routinely rejected by SHPO. Use restoration lines or pre-clear with SHPO.
5. **ADA + Section 106 conflict ignored.** Some historic spaces literally cannot achieve ADA without envelope alteration. Alternative-compliance path requires earlier engagement.
6. **Vibration monitoring missing.** Required by many NPS units + state SHPOs; budget $250–$450/day for the duration of demo + heavy MEP.
7. **Original fixtures discarded.** Some original fixtures (cast-iron lavs, pull-chain water closets) have museum or resale value. Retain unless SHPO authorizes disposal.
8. **Tile mockup not built.** SHPO commonly requires a 4'x4' mockup panel approved BEFORE production install. Schedule + cost.
9. **Plaster patch sub-contracted to drywall sub.** Drywall mud does not match plaster. Use specialty plaster sub; this is a hard SHPO requirement.
10. **Penetrations through historic envelope unauthorized.** Every new exhaust vent + supply run penetration requires SHPO approval. Pre-design all routings + submit with the pre-construction package.
