# Scope Template — Multi-System Gymnasium Renovation

> **Source context:** [`bids/leroy-moore-gym-PV-0749-PV-0753/`](../../bids/leroy-moore-gym-PV-0749-PV-0753/) — PVAMU Leroy Moore Gymnasium racquetball court conversion (PV-0749 / PV-0753), TX state CSP via SSC. The actual Leroy Moore scope is a focused sub-scope (3 racquetball courts → instructional space, ~126 cal days). This template generalizes to multi-system gymnasium renovations: flooring + HVAC + lighting + painting + ADA + (often) bleachers + scoreboard. Most ISD bond programs + university PE-facility refresh projects hit several of these systems simultaneously.
>
> **Typical CSI divisions touched:** 01, 02, 03 (concrete repair), 05 (steel — structural verifications for new RTU loads), 06 (carpentry — bleacher framing), 07 (envelope reseal where roof penetrations for new RTU), 09 (flooring — sport floor primary; paint — high-bay walls; ceilings), 10 (signage, scoreboard mounts, telescoping bleachers), 11 66 23 (gym equipment), 11 68 13 (athletic + recreational equipment), 21 (sprinkler mods for new bleachers / lights), 22 (drinking-fountain replacement, restroom-adjacent if scope), 23 (HVAC — RTUs + ductwork for high-volume), 26 (lighting, scoreboard power, emergency egress), 27 (AV, scoreboard, sound).

## 1. Scope outline (standard sequencing)

| # | Phase | Trade(s) | Typical duration (single ~10,000 SF gym) |
|---|---|---|---|

| 3 | Selective demolition (existing flooring + base; existing lighting; existing RTUs if scope) | Demo sub | 1–2 weeks |
| 4 | Structural verifications (existing roof structure for new RTU loads; existing wall structure for bleacher attachments) | Structural engineer pre-construction | (concurrent w/ demo) |

| 7 | Wall prep + paint (CMU sealer + acrylic on walls; deck paint on ceiling structure) | Paint sub | 1–2 weeks |

| 9 | Bleachers reinstall (telescoping refurb) OR new bleachers install | Specialty bleacher sub | 1 week |

| **Total** | | | **10–16 weeks** |

**Scope-specific variants:**
- **Court-conversion (Leroy Moore-class):** racquetball/handball/squash court → instructional or fitness space; drop scope items 9/12; substitute drywall partition + carpet/LVT. ~8–12 weeks.
- **Sub-scope refresh (paint + lighting only):** ~3–5 weeks; can be completed during academic break.
- **Full bond-program gym overhaul (ISD bond):** add bleachers + scoreboard + AV + new entry vestibule; 16–22 weeks; often combined with new HVAC.

## 2. Standard takeoff-item list

### 2.1 Demolition + removals

```
DEMO
  □ Existing sport floor (SF) — note system (maple over sleeper, synthetic, rubber)
  □ Existing wall base (LF)
  □ Existing wall pads (LF + SF) — if any
  □ Existing acoustic wall panels (SF) — if any
  □ Existing ceiling-mounted equipment (each) — goals, divider curtain, sound, scoreboard
  □ Existing high-bay lighting (each)
  □ Existing RTUs (each) — if scope; coordinate crane + roof patch
  □ Existing ductwork (LF) — if scope
  □ Existing sprinkler heads (each) — only if relocating
  □ Existing telescoping bleachers (each unit) — bag hardware + photo-doc rows
  □ Existing wall paint (SF) — wash + prime if not full repaint
  □ Game-line removal from existing floor (SF) — if floor recoat scope only
  □ Disposal (CY)
```

### 2.2 Sport flooring

```
SPORT FLOORING — PICK ONE per spec
  Maple (premium — basketball, volleyball; ROBBINS, AACER, Connor)
    □ Subfloor system (sleeper-on-vapor-barrier OR floating modular OR poured)
    □ #1 grade NHFR or NTFR northern hard maple, T&G strip (typ 1.5"x2.25" or 2.25"x2.25")
    □ Sand + seal + game-line paint + 2 coats urethane
    □ Mascot logo (size + colors per spec)
    □ Edge expansion ventilation
    □ Wall vent base (LF)
  Synthetic urethane (multi-purpose; Mondo, Gerflor)
    □ Substrate prep (moisture barrier + leveling)
    □ Pad layer (typ 4mm pre-fab pad)
    □ Urethane wear layer (typ 6-8mm)
    □ Game-line stencil + paint
    □ Manufacturer install crew (typ required for warranty)
  Rubber sport tile / sheet (fitness, weight-room, less-formal gym)
    □ Subfloor prep
    □ Tile OR sheet rubber (typ 5-12mm)
    □ Edge ramps + transitions
  Court refinish (re-coat, NOT replace) — alt for shorter scope
    □ Light sand
    □ Re-paint game lines
    □ 2 coats urethane finish
    □ Cure time 5-7 days; gym off-line
```

### 2.3 HVAC

```
HVAC (gymnasium loads are high — verify per RTU brand sizing)
  □ Demo existing RTU (each, by tonnage)
  □ Roof-patch + curb modification (each RTU)
  □ New RTU (each, sized for gym occupancy + air-change per ASHRAE 62.1)
  □ Roof curb (each, factory-fabricated to match new RTU footprint)
  □ Supply ductwork (LF, by size) — typ high-bay with throw-distance calc
  □ Return ductwork (LF, by size)
  □ Diffusers (each) — typ throw-distance type for high-bay
  □ Return grilles (each)
  □ Outside-air damper actuator (each) — for code-required vent rates
  □ DDC controls integration to existing BMS (if applicable)
  □ Test + balance (LS by certified TAB firm)
  □ Crane mobilization + de-mob (LS)
```

### 2.4 Lighting

```
LIGHTING (high-bay LED)
  □ Demo existing fixtures (each)
  □ New LED high-bay (each) — typ 20K-40K lumen; verify glare control (sport lighting)
  □ Photometric calc per IESNA RP-6 (sport lighting)
  □ Mounting bracket / pendant / chain (per fixture)
  □ Sport-grade impact protection (wire guard OR mesh guard) (each)
  □ Lighting controls (occupancy sensor + dimming + scheduling)
  □ Emergency egress fixtures (each) — typ 90-minute battery backup
  □ Exit signs (each) — code-required at every exit + path
  □ Scoreboard power circuit (LS — typ 208V/30A dedicated)
  □ Sound-system power circuit (LS)
  □ Re-circuit if existing panel undersized
```

```

  □ CMU sealer (block filler) (SF wall)
  □ Acrylic semi-gloss above wainscot (SF) — typ 2 coats over primer

  □ Ceiling deck paint (SF) — high-build acrylic; typ light gray or white

  □ Accessible-seating row markings + designations (each)

  □ Wall pads (LF + SF) — typ 6' high, padded vinyl-faced
```

## 5. Common alternates / VE opportunities

| Item | Base | Alternate | Trade-off |
|---|---|---|---|
| Floor system | Maple sport floor | Synthetic urethane OR rubber | Synthetic is 25–40% cheaper; rubber is 50–60% cheaper (fitness scope only); maple is the premium for basketball/volleyball |
| Floor scope | Replace | Refinish (sand + recoat) | Refinish is 75–85% cheaper IF existing floor sound; verify subfloor moisture |
| Lighting | LED high-bay w/ control | LED high-bay w/o control | Controls save energy long-term but add $80–$150 per fixture upfront |
| RTU sizing | Per ASHRAE 62.1 strict | Per as-built occupancy (typ lower) | Right-sizing saves 15–25% on RTU cost; document basis in submittal |
| RTU brand | Trane / Carrier | Daikin / York | Brand spec often locked by district preference |
| Bleachers | New | Refurb existing | Refurb is 50–70% cheaper IF tracks + structure sound |
| Bleacher operator | Manual | Power-fold | Power adds $4K–$8K per unit; verify staff need |
| Wall pads | Padded vinyl | None (paint only) | Skip pads at $85–$140/LF if no contact-sport use |
| Goals | Ceiling-suspended power | Wall-mount manual | Manual is 60–80% cheaper but slower to deploy/store |
| Scoreboard | New | Existing reinstall | Reuse saves $4K-$12K if existing is functional |
| AV / sound | Full new | Sound-system upgrade only | Skip scoreboard if existing works |

**Critical-path items:**
- **Sport floor cure time** (5–7 days post-finish before any traffic; maple installs may require 3–5 weeks total floor-down)
- **RTU lead time** (8–14 weeks for custom units; 4–8 weeks for stock)
- **Bleacher fabrication** (10–16 weeks for new telescoping; 4–8 weeks for refurb parts)
- **Scoreboard custom logo / mascot** (6–10 weeks)
- **Academic calendar lock** (summer build window is the hardest deadline — start-of-school must hit)

## 7. Code references

- **IBC Chapter 10** (Means of Egress) — gym occupancy load + exit width
- **IBC 1009** (Accessible Means of Egress) — accessible seating + path
- **IBC 1029** (Bleachers / Folding-and-Telescopic Seating + Grandstands) — bleacher attachment + spacing + sightlines
- **ADA 221** (Assembly Areas) — wheelchair spaces + companion seats; line-of-sight
- **NFPA 13** (Sprinkler) — coverage updates if ceiling structure modified
- **NFPA 70 / NEC** — sport-grade lighting + scoreboard power
- **IECC** — energy code (LED + HVAC efficiency)
- **IMC** — HVAC ventilation rates
- **ASHRAE 62.1** (Ventilation for Acceptable IAQ) — gym occupancy is high-CFM
- **IESNA RP-6** (Sports + Recreational Area Lighting) — photometric requirements by sport class
- **ASTM F2772** (Sport Floor Performance) — shock absorption + ball rebound + friction
- **ANSI/MFMA PUR-1** (Maple Floor Specification) — for maple sport floor
- **NCAA + NFHS** sport-specific dimensional requirements (basketball court size, volleyball court, etc.) — for collegiate / high-school competition gyms
- **Local AHJ** — bleacher capacity inspection + occupancy posting

## 8. Common pitfalls

1. **Sport-floor sub-contracted to commercial flooring sub.** Sport flooring requires manufacturer-certified installer for warranty. Verify cert before subbing.
2. **Cure time underestimated.** Maple sport floor + urethane finish needs 5-7 days minimum cure post-finish before any traffic; full hardness 30 days. Schedule academic break accordingly.
3. **HVAC sized for as-built occupancy, not code.** Gym occupancy load drives ventilation rate. Right-sizing saves cost but must comply with ASHRAE 62.1 + IMC; document basis.
4. **Roof structure not verified for new RTU loads.** New RTU is often 20-40% heavier than existing. Pre-construction structural verification + stamped letter required.
5. **Lighting glare not analyzed.** Sport-grade LED high-bay needs photometric calc + glare control (cutoff angle, frosted lens, or shielded). Bare LED creates "hot spot" glare unacceptable for basketball/volleyball play.
6. **Bleacher load + attachment not verified.** Wall-attached telescoping bleachers transfer significant moment to the wall. Engineer verification critical; commonly missed.
7. **Acoustic treatment forgotten.** Gym acoustic environment is brutal; spec'd acoustic panels often value-engineered out, leading to facility complaints post-occupancy. Push back hard if VE proposed.
8. **Game lines + logo specs incorrect.** NFHS / NCAA / NJCAA + local district have varying line specs (3-point arc varies). Verify per intended sport + competition level.
9. **Scoreboard custom logo lead-time missed.** Custom mascot logos take 6-10 weeks from order; under-budget at proposal stage = punch-list item at closeout.
10. **Existing floor moisture not tested.** New sport flooring requires substrate moisture <4% (CaCl) or <75% RH. Missing the test = warranty void.
11. **Drinking fountain ADA hi/lo missed.** Older gyms have non-ADA fountains; replacement to ADA hi/lo bottle-filler often forgotten in scope.
12. **Academic calendar buffer too thin.** Summer build window has ZERO float at the end. Build in 2-3 wk buffer for floor cure + bleacher commissioning + AHJ inspection.
13. **Owner-furnished AV / goal scope ambiguous.** Owner-provided goals + scoreboards commonly delivered late or wrong-spec; coordinate scope-split early.
