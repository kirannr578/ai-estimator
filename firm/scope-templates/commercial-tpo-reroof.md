# Scope Template — Commercial Low-Slope Re-Roof (TPO / EPDM / PVC)

> **When to use this template:** any commercial / institutional low-slope (≤ 2:12) re-roof RFP
> where the system specified is a single-ply thermoplastic (TPO, PVC) or thermoset (EPDM)
> membrane over rigid-board insulation + cover board. Covers mechanically-attached,
> fully-adhered, and ballasted attachment methods; tear-off and overlay (recover) paths;
> insulation upgrade-to-current-code; and the perimeter sheet-metal / edge-metal / coping /
> scupper / drain / RTU-coordination scope that wraps every commercial re-roof. **NOT a fit for:**
> historic / NRHP-listed structures (use [`roof-repair-historic.md`](roof-repair-historic.md)),
> standing-seam metal re-roofing, or steep-slope shingle work.
>
> **Source bid that informed this template:** [`bids/tmd-camp-maxey-roof-2026-0626/`](../../bids/tmd-camp-maxey-roof-2026-0626/)
> — TMD Camp Maxey Bldg. 1 dual-roof (lower mech-fastened + upper fully-adhered) 60-mil TPO
> tear-off + new system, with full perimeter sheet-metal scope. Buy-American + Davis-Bacon
> overlay applied (federal-funded state-agency channel via private prime).

## 1. Typical scope summary

A commercial low-slope re-roof RFP almost always packages the same six work elements; the
estimator's job is to map the project-specific spec into them and to call out the
inclusion/exclusion lines that the spec leaves ambiguous.

| # | Work element | Typical spec language | Decision drivers |
|---|---|---|---|
| 1 | **Tear-off vs. recover (overlay)** | "Remove existing roof system down to deck" vs. "Install new system over existing where existing is sound" | Existing system condition + IBC 1511 (max 2 layers) + deck moisture survey + insulation upgrade required by current IECC |
| 2 | **Membrane system** | "60-mil TPO" / "60-mil EPDM" / "60-mil PVC", attachment method (**mechanically-attached** / **fully-adhered** / **ballasted**), color, manufacturer | Wind-uplift calc (FM 1-60 / 1-75 / 1-90 / 1-105) + deck type + adhesive temperature window + reflectivity (Title 24 / ENERGY STAR cool-roof) |
| 3 | **Insulation + cover board** | Polyiso (LTTR R-value to meet IECC), HD ISO or gypsum cover board, attachment method matches membrane | IECC climate-zone R-value floor (R-25 / R-30 / R-35) + cover-board hail-rating (FM SH or VSH) + tapered-insulation slope-to-drain |
| 4 | **Edge metal / coping / scuppers / drains** | 24-ga prefinished metal drip edge, coping cap, gravel stop, through-wall scuppers, overflow scuppers, new roof drains + bowls + clamps | ANSI/SPRI ES-1 edge-metal wind-uplift compliance + IPC 1108 drain sizing + scupper-to-overflow code (IPC 1108.3) |
| 5 | **RTU + penetration coordination** | "Disconnect + re-set existing rooftop equipment" / "TPO wrap on all vent stacks" / "New pipe boots" | EA count of RTUs + curbs + vent stacks + skylights + hatches + lightning-protection bases; HVAC sub for disconnect-reconnect; electrical for any cable management |
| 6 | **Manufacturer system warranty** | "20-year NDL system warranty" (typ 15 / 20 / 25 / 30-yr options) | Manufacturer requires factory-certified installer + post-install inspection by mfr rep before warranty issues; influences sub selection |

**BPC self-perform fit:** roofing tear-off + new membrane install is within BPC's self-perform
lane (`firm/firm-profile.md` §9). Pre-qualify against **manufacturer's certified-installer**
list before bidding any project where the warranty term is > 15 years — Carlisle SynTec
Pro Roofing Advantage, Firestone (Holcim) Master Contractor, GAF Master Select, JM Peak Advantage,
Sika Sarnafil Roofing Excellence, Versico VIP, Mule-Hide Premium. First-of-kind exposure
without certification is a real risk on a 20-yr+ NDL warranty.

## 3. Standard scope sequencing

| # | Phase | Trade(s) | Typical duration (10,000 SF roof) | Typical duration (40,000 SF roof) |
|---|---|---|---|---|
| 1 | Pre-construction: roof condition survey + deck moisture survey (infrared or capacitance) + wind-uplift calc + submittals (membrane data sheet, ISO data sheet, fastener pattern, edge-metal SPRI ES-1) | Roofing contractor + manufacturer's tech rep | 1–2 weeks | 2–3 weeks |
| 2 | Mobilization: dumpster + crane / boom-lift + material drop (pallet staging on roof OR ground) + safety perimeter / OSHA 1926 Subpart M fall protection | GC | 2–3 days | 3–5 days |
| 3 | RTU + roof-equipment disconnect (HVAC sub coordinates with owner for shutdown window) | HVAC + electrical subs | 1 day per cluster | 2–3 days |
| 4 | Tear-off — gravel / ballast first, then membrane + insulation down to deck; spot deck repair as exposed | Roofer | 3–5 days | 1–2 weeks |
| 5 | Deck inspection + repair (rotted plywood, rusted steel deck, spalled lightweight concrete) — allowance-based change order if > spec | Carpenter + roofer | parallel | parallel |
| 6 | Air/vapor barrier on deck (if spec'd) | Roofer | 1–2 days | 3–5 days |
| 7 | Insulation install — base layer (typ. 2 layers staggered for thermal break) + tapered crickets at drains + HD cover board | Roofer | 3–5 days | 1.5–2 weeks |
| 8 | Membrane install — mech-attached (fastener + plate per pattern) OR fully-adhered (low-rise foam or bonding adhesive) OR ballasted | Roofer | 3–5 days | 1.5–2 weeks |
| 9 | Flashings — wall + curb + penetration + scupper + drain + expansion joint | Roofer | 3–5 days | 1 week |
| 10 | Edge metal + coping + gutter + downspouts (SPRI ES-1 compliant) | Sheet-metal sub | 2–3 days | 1 week |

## 4. Quantity-takeoff hooks (fields to capture at site visit)

- [ ] **Roof field area (SF)** — net of penetrations + curbs > 4 SF; broken out per roof section (lower / upper / north / south) where attachment method differs
- [ ] **Perimeter LF** — drives drip edge, coping, fascia, gravel stop, perimeter plywood / blocking, perimeter-zone fastener pattern (typ 50% denser than field per FM 1-90)
- [ ] **Corner-zone area (SF)** — corners take a 3rd densification per FM uplift; capture corner LF × 10% of building width as a separate takeoff line
- [ ] **Wall + curb flashing LF** — drives 60-mil membrane flashing material (sold as separate accessory)
- [ ] **Curb count + height (EA)** — RTU curbs, skylight curbs, hatch curbs; drives flashing labor (typ 1 hr per curb per crew member)
- [ ] **Penetration count by type (EA)** — vent stacks (round) → pre-formed pipe boot; square / rectangular penetrations → field-fabricated wrap or pitch-pocket; lightning-protection bases → seal-on-membrane detail
- [ ] **RTU count + tonnage / weight (EA)** — drives HVAC-sub disconnect-reconnect line + crane cost if RTUs must be lifted off curbs
- [ ] **Drain count + size (EA, dia)** — new bowls + clamping rings + strainers; verify drain locations match low-points of tapered ISO
- [ ] **Scupper count + size (EA, primary + overflow)** — through-wall scupper boxes + downspout heads; overflow scuppers per IPC 1108.3
- [ ] **Gutter LF** — K-style or box gutter; size per IPC drainage calc
- [ ] **Downspout count + LF** — typ 3"×4" rect or 4" round; splash blocks or tie-in to storm
- [ ] **Edge metal LF (drip edge + gravel stop + coping)** — verify SPRI ES-1 compliance for the wind zone
- [ ] **Hatch count (EA) + ladder access** — roof hatches, ship's ladder, fixed ladder
- [ ] **Walkway pad LF** — at HVAC equipment, drain paths, hatch egress (mfr's pad system)
- [ ] **Deck type by section** — steel deck (22-ga / 20-ga / 18-ga), lightweight insulating concrete (LWIC), structural concrete, plywood (existing), gypsum deck — drives fastener selection + adhesive applicability
- [ ] **LWIC thickness (where applicable)** — drives mechanical-fastener length spec; brittle / saturated LWIC may fail pull-out
- [ ] **ASCE 7 wind zone** — V_ult mph + exposure category (B / C / D); drives uplift calc + attachment pattern
- [ ] **Building height + parapet height** — drives counterflashing termination + lift / scaffold sizing + edge-metal cleat schedule
- [ ] **Existing insulation thickness + condition** — drives whether tear-off or recover is permissible (IBC 1511 max 2 layers)
- [ ] **Roof access (interior stair / exterior ladder / hatch)** — drives material-handling plan (crane to roof vs. boom-lift vs. interior hoist)
- [ ] **Staging area available** — dumpster placement + material drop + crane swing radius

## 5. Inclusions / exclusions checklist (lift verbatim into the quote)

### 5.1 Standard inclusions

```
INCLUDES
  ✓ Roof condition survey + deck moisture survey (infrared or capacitance) — single pass
  ✓ Tear-off of existing membrane, insulation, and edge metal to the structural deck
  ✓ Disposal of tear-off material at a permitted C&D landfill; dumpster + haul incl.
  ✓ New roof insulation per spec'd R-value (LTTR), in 2 staggered layers, mechanically
    fastened or fully adhered per attachment spec
  ✓ New 1/2" HD ISO (or gypsum) cover board over insulation
  ✓ New 60-mil membrane per spec'd manufacturer + attachment method, with mfr-approved
    fasteners, plates, and adhesives
  ✓ New 60-mil membrane flashings at all walls, curbs, and penetrations
  ✓ Pre-formed pipe boots at all round vent stacks; field-fabricated wrap at irregular
    penetrations; pitch-pocket assemblies at clusters per mfr detail
  ✓ New 24-ga prefinished metal edge metal (drip edge OR gravel stop), coping cap,
    counterflashing, gutter, and downspouts per spec — color per owner selection
  ✓ New roof drain clamping rings + strainers at existing drain locations
  ✓ Disconnect + reconnect of existing RTU power + condensate per owner-furnished MEP
    cut sheets (excludes any RTU replacement)
  ✓ Walkway pads at hatch egress + drain paths + RTU service zones per mfr's standard
  ✓ Manufacturer's certified-installer labor with end-of-job mfr inspection for warranty
  ✓ Manufacturer's NDL system warranty (term per spec — typ 15 / 20 / 25 yr)
  ✓ Workmanship warranty (typ 2-yr from substantial completion)
  ✓ Daily cleanup + final magnet sweep + closeout documentation (O&M, warranty cert, as-builts)
```

### 5.2 Standard exclusions (call out explicitly in the quote)

```
EXCLUDES (additional cost via change order if encountered)
  ✗ Structural deck repair beyond [N] SF allowance — replace rotted plywood, rusted
    steel deck, or spalled LWIC found during tear-off; priced at $___/SF additive
  ✗ Asbestos abatement — if existing BUR / mod-bit contains ACM (asphalt mastic, felt,
    cementitious flashing), a separate licensed abatement sub is required; not in this scope
  ✗ Lead-paint abatement on existing flashings or scuppers — separate scope if encountered
  ✗ Insulation upgrade beyond spec'd R-value (e.g., to meet a future IECC version) — base
    bid is per spec; upgrades priced as alt
  ✗ Tapered insulation crickets beyond those required to direct flow to existing drains —
    re-design to add slope or relocate drains is excluded
  ✗ Roof structural reinforcement for new RTU loads — base bid assumes existing structure
    supports re-set RTUs as-is
  ✗ Roof drain piping below the roof line — plumbing rework, leader piping, conductor
    boxes, storm tie-ins are excluded
  ✗ Lightning-protection take-down + re-install — UL Master-Label specialty sub required;
    priced separately if existing system is present
  ✗ Interior repairs (water-damaged ceiling tile, drywall, paint) below leaking sections
    of existing roof — owner-directed; separate change order
  ✗ Photovoltaic array take-down / re-install — separate PV contractor required
  ✗ HVAC equipment replacement or relocation — only disconnect + reconnect of existing RTUs
  ✗ Snow / ice removal during installation if weather demands — priced T&M if encountered
  ✗ Permits + plan-review fees — owner-furnished unless specified
  ✗ Bonds (P&P, payment) — priced separately at 1.5–2.5% of contract value if required
  ✗ Sales tax — flag if project is tax-exempt or if sub-PO is in a tax-exempt agency channel
```

## 6. Common risks (initial register — re-score after site visit)

| # | Risk | Likelihood | Impact | Mitigation |
|--:|---|:--:|:--:|---|
| 1 | **Tie-in to existing adjacent roof** at a phase boundary or to a roof that is NOT in scope — leak path at the seam; warranty boundary unclear | M | H | Pre-construction conference with mfr's tech rep; document tie-in detail in shop drawings; photograph existing condition before tear-off; carry the tie-in detail as an explicit submittal |
| 2 | **Weather window** — TPO / PVC seam welding has temperature minimums (typ. 40°F substrate); EPDM adhesive has dew-point + temperature limits; extreme summer heat (> 95°F) affects worker safety + adhesive open time | H | M | Build weather days into schedule (TX summer: 1–2 days/week; gulf-coast spring/fall: 2–3 days/week for rain); price a winter-condition premium if NTP is Nov–Feb |
| 3 | **Deck moisture survey not done before tear-off** — wet insulation hides deck damage; saturated LWIC fails fastener pull-out; wet steel deck rusts under new membrane | M | H | Mandate IR or capacitance survey in pre-construction; price a deck-repair allowance ($N/SF for plywood patch, $N/SF for steel-deck panel replacement) as a separate quote line |
| 4 | **Asbestos in existing built-up roofing** — gravel-surfaced BUR pre-1980s often has ACM in mastic, felt, or cementitious flashing; tear-off without abatement is a federal NESHAP violation | M | H | Require owner-furnished asbestos survey before tear-off; if not provided, sample + lab-test 3 cores at BPC's cost ($300–$800) and price abatement as separate allowance |
| 5 | **Davis-Bacon prevailing wage** — federal-funded or state-agency projects with DBA add 20–40% to roofer + sheet-metal-worker labor cost; weekly WH-347 certified payroll burden | H | H | Pull current county WD from sam.gov before pricing; configure WH-347 process at sub-PO sign, not at NTP; add 1–2% admin uplift to GC line |
| 6 | **Buy American Act compliance** — federally-funded projects (FAR 52.225-9, BABA, 41 U.S.C. 10) require domestic-origin attestation on iron + steel + manufactured products; TPO membrane + ISO + plywood + edge metal all in scope | M | M | Confirm with mfr / supplier in writing **before** sub-PO; document Buy-American certificate at material delivery; bias supplier selection toward domestic manufacturers (Carlisle, Firestone/Holcim, GAF, JM, Versico) |
| 7 | **Wind-uplift calculation not provided** — spec references uplift calcs as the basis for fastener pattern + adhesive coverage; calcs frequently absent from the drop | M | M | Request from EOR; if not provided, use mfr's standard FM-1-75 (typical) or FM-1-90 (high-wind) pattern with caveat in the quote; document the assumption |
| 8 | **Manufacturer certification requirement** — > 15-yr NDL warranty requires certified-installer + mfr rep inspection; BPC's first commercial-TPO project may not qualify for 20-yr+ NDL | M | M | Apply for installer certification ahead of bid (typ. 30-day approval + 1-project shadow); offer 15-yr warranty as a base + 20-yr as a priced alt contingent on cert |
| 9 | **RTU coordination + facility shutdown windows** — disconnecting RTUs requires facility cooperation; some sites (data centers, hospitals, military comms) cannot tolerate any shutdown | M | M | Confirm shutdown windows at site visit; price after-hours / weekend premium if owner cannot allow daytime shutdowns; coordinate HVAC sub for short outage windows |
| 10 | **Lightweight insulating concrete (LWIC) condition** — older roofs with LWIC fill over steel deck may be spalled, brittle, or saturated; pull-out values fail uplift calc | M | M | Probe at site visit; price a LWIC repair allowance ($N/SF for tear-out + new LWIC pour OR replace with rigid insulation); flag in the quote |
| 11 | **Edge metal SPRI ES-1 compliance** — ANSI/SPRI ES-1 wind-uplift testing for perimeter edge metal is required by IBC 1504.5; older details may not comply | L | M | Use mfr's pre-engineered SPRI ES-1-tested edge-metal system rather than field-fabricated profile; provide test report as submittal |
| 12 | **Photovoltaic array on roof** — PV de-mount + re-mount specialty sub; tie-in to membrane through a manufacturer-approved mount system (S-5, EcoFasten, PMC); not in this scope | M | M | Confirm at site visit; flag PV as a separate trade; document existing condition photos before tear-off |
| 13 | **Long-lead membrane or accessory** — non-stock colors (tan, gray, custom) often 4–8 weeks; high-reflectivity (Title 24) TPO sometimes longer; mfr-specific accessories (drain assemblies, edge metal) 4–12 weeks | M | M | Order critical-path material on sub-PO award; do not commit to a mob date that assumes stock availability |
| 14 | **Insurance + COI status** — commercial roofing GC requires CGL ($1M/$2M), Auto ($1M CSL incl. Hired/Non-Owned), WC statutory, with prime/owner as Additional Insured | H | H | Verify current COI before quote submission; ensure broker can issue AI endorsement within 5 business days of sub-PO |
| 15 | **Submission channel single-point-of-failure** — many sub-PO bids are by email (gmail, personal-domain) with no formal receipt process | M | M | Send at least 4 hr before the deadline; cc internal bid lead; phone-confirm receipt 30 min after sending |

### 7.3 Escalation + labor sourcing

- **Material escalation** — track **AGC Construction Inflation Alert** and **ENR Construction
  Cost Index (CCI)** quarterly; TPO + polyiso are petroleum-derivative materials and move
  with crude-oil + MDI chemistry feedstocks. Apply 4–8% material escalation for any project
  with > 6 months between bid and material order; 8–15% for > 12 months.
- **Labor — non-DBA market rate** — pull current local roofer + sheet-metal-worker wage
  from **BLS OEWS (Occupational Employment & Wage Statistics, SOC 47-2181 Roofers + 47-2211
  Sheet Metal Workers)** for the project MSA; OEWS publishes annual mean + 10/25/50/75/90
  percentile rates. BPC's standard fully-burdened multiplier is ~1.65× on top of base wage
  (FICA + WC + UI + benefits + indirect).
- **Labor — DBA / Davis-Bacon** — pull current county / state building-construction
  Wage Determination from `https://sam.gov/wage-determinations` and apply per BLS SOC
  classification mapping; track WD revisions during the project (apply most-recent at
  contract award per 29 CFR 5.5(a)(1)(i)). Add 1–2% to GC line for WH-347 weekly certified
  payroll administrative burden.
- **TPO-specific lead-time tracking** — Carlisle SynTec, Holcim/Firestone, GAF, and JM
  publish monthly lead-time updates to their distributor networks; have the distributor
  forward the current lead-time bulletin before bid lock.

## 8. Common pitfalls

1. **Tear-off depth assumed; spec ambiguous.** Spec may say "remove loose gravel" when the actual
   intent is full tear-off down to deck. Confirm in writing before pricing — the labor
   delta is 5–10× between "gravel skim" and "tear-off to deck."
2. **Insulation upgrade implied but not specified.** IECC + many state energy codes require an
   insulation upgrade on re-roof (commercial code triggers at most full-tear-off events).
   Owner may not know; verify with AHJ.
3. **Edge metal wind-uplift class not specified.** IBC 1504.5 requires SPRI ES-1; bid with
   pre-engineered ES-1 system (not field-fab profile) to avoid an inspection-failure rework.
4. **RTU disconnect-reconnect coordination missed.** Each RTU on the roof adds 2–4 hr of HVAC-
   sub labor + a facility-shutdown window. Count RTUs at site visit, not from drawings.
5. **Lightning protection on roof not surfaced.** Existing LP system (air terminals + down
   conductors on parapet + ground straps) must be take-down + re-install by UL Master-Label
   sub; specialty trade. Inspect at site visit; price separately.
6. **Asbestos in BUR not flagged.** Pre-1980s gravel-surfaced built-up roofing frequently
   contains ACM in mastic, felt, or cementitious flashing. NESHAP requires owner-furnished
   survey + licensed abatement. Sample at BPC's cost ($300–$800) if owner survey is missing.
7. **Photovoltaic array de-mount / re-mount underscoped.** Solar-equipped roofs require a
   PV-specialty sub to de-mount + reseal mounting points + re-mount — adds $8K–$25K per
   array even on small systems. Verify at site visit.
8. **Buy American documentation chain not set up at sub-PO.** Each material delivery must
   carry a domestic-origin attestation; missing one at a federal audit can void payment.
   Set up the attestation log at sub-PO sign, not at NTP.
9. **Wind-uplift "default" pattern assumed without check.** FM 1-90 is a typical commercial
   assumption but coastal sites (TX gulf, FL, NC) routinely require FM 1-105 or higher.
   Pull the project-specific ASCE 7 wind data before committing to a fastener pattern.
10. **Manufacturer-rep final-inspection scheduling slips.** NDL warranty cannot be issued
    until mfr rep walks the completed roof and signs off; rep schedules typically need
    10–15 business days notice. Build into the closeout schedule, not the punch list.
11. **Davis-Bacon scope mis-classification.** Roofers (SOC 47-2181) and sheet-metal workers
    (SOC 47-2211) are different DBA classifications with materially different wage rates;
    misclassifying drives DOL audit risk. Map at sub-PO sign.
12. **Workmanship warranty start date** — confirm in writing whether warranty starts at
    substantial completion or at owner acceptance; DBA / federal projects may delay the
    acceptance date past the install date.
